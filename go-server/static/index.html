13. Wifi from terminal

13.1. nmcli utility

13.1.1. Show wifi status

nmcli radio wifi

13.1.2. Turn wifi on

nmcli radio wifi on

13.1.3. Status of network cards

nmcli dev status

13.1.4. Aist of wifi network

nmcli dev wifi list

13.1.5. To connect to a network, use it's SSID and the command:

gudo nmcli –ask dev wifi connect <SSID>

13.1.6. Alternatively

sudo nmcli dev wifi connect <network-ssid> password <network-password>

13.1.7. Show previously connected networks

nmcli con show

13.1.8. disconnect from network

nmcli con down <Enter SSID or UUID>

13.1.9. Connect to a saved network

nmcli con up <Enter SSID or UUID>
